GUARD / WORKER / USER APP FUNCTIONS

SCAN QR CODE

Client request for QR code is generated online and sent to client or service providers who will in turn print the codes and paste the QR codes in various locations. Workers login to the app with their username provided to go around and scan these QR codes. For every scan, username, time of scan, type and exact location of scan is captured by the solution.

INCIDENT REPORT

If there is any incident or anything else to report to the supervisor or manager, user can open Incident report - capture a picture and put some notes and post it which will be immediately visible to the supervisor on the web admin center or admin app instantly.If there is any incident or anything else to report to the supervisor or manager, user can open Incident report - capture a picture and put some notes and post it which will be immediately visible to the supervisor on the web admin center or admin app instantly.

OFFLINE SCANNING

Even if data is exhausted in the device used by guards or workers, the app will store the scans internally on the device and will deliver it to the solution once there is device gets connected again with data or wi- fi. Don't worry about losing a scan or patrol just because we didn't load airtime on-time Our mobile development services include building mobile native application across all of the major mobile platforms.

CHECK-IN/CHECK-OUT

Users can use the check-in/check-out option to check- in to a location and check-out with a selfie. All check-in and check-out actions are captured with photo, time and exact location which can be plotted on the map for analysis

TAG MY LOCATION

Users can use 'Tag my Location' option to tag their current location for reference purposes if there is no QR code to scan for a specific location during special circumstances or during incidents. These scans are captured as 'CLOSED DOOR' action which is visible in the admin

BEAT BOOK

Beat Book is used for automatic scans. Beat book can only be used if all the QR codes initially pasted in various locations are captured in the system and linked to a specific site and also location. If this setting is performed, then users can simply open Beat Book and press the corresponding location which will be captured if the user is less than 30 meters of QR location setting
